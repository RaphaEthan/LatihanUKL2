Fungsi
Program ini berfungsi sebagai penentu bilangan ganjil atau genap. Tujuannya adalah untuk mengidentifikasi apakah sebuah bilangan bulat yang dimasukkan oleh pengguna adalah bilangan genap atau ganjil.

Cara Kerja
Program akan meminta pengguna untuk memasukkan sebuah "bilangan".

Pengguna memasukkan satu angka (disimpan dalam variabel bilangan).

Program akan segera memeriksa sisa bagi (modulus) dari angka tersebut ketika dibagi 2 (menggunakan operasi bilangan % 2).

Logika Keputusan:

Jika sisa bagi adalah 0, program akan mencetak [bilangan] adalah genap.

Jika sisa bagi bukan 0 (atau 1), program akan mencetak [bilangan] adalah ganjil.
